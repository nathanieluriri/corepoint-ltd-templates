# Multi Adaptation Plan

## Positioning Fit
- Use Multi as the most traditional corporate option for Core Point Ltd.
- Preserve its classic multi-section business layout, carousel hero, service grid, FAQ, team, and contact structure.
- Shift it from a generic company template into a credible management consulting homepage with layered institutional messaging.

## Template-Specific Direction
- Visual tone:
  polished corporate, light-background dominant, crisp navy-charcoal base, warm neutral sections, and selective accent color for emphasis.
- Design memory:
  a disciplined boardroom-ready site that feels trustworthy and established without becoming stiff.
- Constraint:
  keep the original section ordering and strong content density, but sharpen hierarchy and reduce stock-template repetition.

## File Scope
- `Multi/index.html`
- `Multi/assets/css/main.css`
- `Multi/assets/js/main.js`
- `Multi/assets/img/*`
- `Multi/forms/contact.php`

## Section TODOs

### Header and Navigation
- Replace site identity with Core Point Ltd logo and name.
- Update page metadata and open graph basics if added later.
- Rework nav labels to reflect consulting structure and remove portfolio-heavy emphasis if it distracts from service discovery.

### Hero Carousel
- Rewrite all three slides so they tell one coherent consulting story instead of three unrelated template headlines.
- Suggested slide themes:
  Strategy into Performance,
  Leadership and Transformation,
  Sustainable Organizational Excellence.
- Replace generic CTA language with consultation and service exploration actions.
- Review carousel pacing and controls so the messaging remains readable, especially on mobile.

### About
- Use the about section for the full firm introduction.
- Tighten the copy into readable blocks that explain:
  who Core Point Ltd serves,
  how it works,
  what outcomes it enables.
- Replace filler bullets with the supplied "Why Core Point Ltd" differentiators.

### Stats
- Remove placeholder counters unless real values exist.
- Repurpose this section into strategic evidence without fabricated metrics:
  client types served, delivery approach, consulting disciplines, or engagement principles.

### Features
- Convert features into the firm's core operating strengths.
- Use each feature item to explain how the company bridges strategy, systems, people, and execution.
- Ensure icons support corporate consulting themes rather than product or digital-only cues.

### Services
- Map the six supplied service categories directly into the service cards.
- Expand service descriptions slightly so they feel senior and outcome-oriented.
- Consider linking service cards to `service-details.html` for future detailed service pages.

### Testimonials
- If there are no real testimonials, replace the section with:
  client engagement philosophy,
  leadership commitments,
  or strategic outcome principles.
- Keep the visual rhythm, but do not publish fabricated quote content.

### Call to Action
- Rewrite the CTA into a consultation prompt:
  help organizations align strategy, strengthen execution, and improve performance.
- Use a stronger value line and a clear action button.

### Portfolio
- Portfolio is misaligned for management consulting if left as image projects.
- Convert it into one of these:
  sectors served,
  consulting focus areas,
  transformation themes,
  or capability case categories with abstract visual tiles.
- Remove category labels that imply design studio or product showcase work.

### Team
- Replace the existing four-person layout with the supplied three leaders.
- Rebalance the grid so three cards look intentional, centered, and premium.
- Support long credential strings, especially for Dr. Uduak Amasi.

### Pricing
- Replace pricing cards with engagement model cards.
- Possible labels:
  Strategic Advisory Engagements,
  Transformation Delivery Support,
  Executive Development Programs.
- Use descriptive language around scope and collaboration rather than price points.

### FAQ
- Insert the supplied seven FAQs and rewrite the intro so it feels executive and informative.
- Check accordion accessibility and spacing for longer answers.

### Contact and Footer
- Replace placeholders with website, email, phone, and office details when available.
- Refocus the form around advisory discovery conversations.
- Update footer links so they reinforce company credibility and service pathways.

## Responsive and Polish TODOs
- Reduce carousel text width and line length for tablet readability.
- Improve card consistency across services, team, and converted portfolio tiles.
- Tighten spacing between repeated light sections so the page feels more premium.
- Review button styling, icon tone, and divider treatments for a more tailored corporate finish.

## Content Risks to Resolve During Implementation
- Portfolio section needs a final replacement strategy.
- Testimonials should not go live without authentic source material.
- Contact details and any future service-detail links still need confirmation.
